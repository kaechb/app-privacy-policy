Privacy Policy
Last Updated: July 1, 2026
This Privacy Policy outlines the data handling practices for this application.
Data Collection and Storage
This application operates locally on your device. It does not collect, record, or transmit any personal data to developer-controlled servers.
Third-Party Services
The application interfaces with the following external APIs strictly to fulfill user-initiated requests:
Anthropic (Claude): Utilized for processing and generating text responses.
Pexels: Utilized for fetching image media.
No user-identifiable tracking data or persistent identifiers are transmitted to these third parties.
Contact
For questions regarding this policy or data practices, please use the contact email provided in the App Store Connect testing details.
